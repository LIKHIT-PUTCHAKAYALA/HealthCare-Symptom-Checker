<h2 style="font-size:32px;">‚öôÔ∏è Setup and Installation</h2>
<p style="font-size:18px;">Follow these steps to get the application running on your local machine.</p>

<h3 style="font-size:28px;">1. Prerequisites</h3>
<ul style="font-size:18px;">
<li>Python 3.8 or higher</li>
<li>pip (Python package installer)</li>
<li>A Google Gemini API Key. You can obtain one from Google AI Studio.</li>
</ul>

<h3 style="font-size:28px;">2. Clone the Repository</h3>
<pre style="font-size:16px;">
git clone https://github.com/your-username/healthcare-symptom-checker.git
cd healthcare-symptom-checker
</pre>

<h3 style="font-size:28px;">3. Set Up a Virtual Environment</h3>
<p style="font-size:18px;">It's recommended to create a virtual environment to manage project dependencies.</p>

<p style="font-size:18px;"><b>On macOS/Linux:</b></p>
<pre style="font-size:16px;">
python3 -m venv venv
source venv/bin/activate
</pre>

<p style="font-size:18px;"><b>On Windows:</b></p>
<pre style="font-size:16px;">
python -m venv venv
.\venv\Scripts\activate
</pre>

<h3 style="font-size:28px;">4. Install Dependencies</h3>
<p style="font-size:18px;">Install all the required Python packages from the requirements.txt file.</p>
<pre style="font-size:16px;">
pip install -r requirements.txt
</pre>

<h3 style="font-size:28px;">5. Configure Environment Variables</h3>
<p style="font-size:18px;">Create a file named .env in the root directory of the project. This file will store your API key securely.</p>
<pre style="font-size:16px;">
GEMINI_API_KEY="YOUR_GEMINI_API_KEY_HERE"
</pre>
<p style="font-size:18px;">Replace "YOUR_GEMINI_API_KEY_HERE" with your actual Gemini API key.</p>

<h3 style="font-size:28px;">6. Run the Application</h3>
<p style="font-size:18px;">Start the Flask development server:</p>
<pre style="font-size:16px;">
flask run
</pre>
<p style="font-size:18px;">Or, you can run the Python script directly:</p>
<pre style="font-size:16px;">
python app.py
</pre>
<p style="font-size:18px;">The application should now be running at http://127.0.0.1:5000. Open this URL in your web browser.</p>

<h2 style="font-size:32px;">üöÄ How It Works</h2>
<ul style="font-size:18px;">
<li><b>User Input:</b> The user enters their age, gender, and a detailed description of their symptoms on the web interface.</li>
<li><b>API Call:</b> The Flask backend receives the data and constructs a detailed system prompt, which includes the user's information and specific instructions for the AI.</li>
<li><b>Gemini API:</b> A request is sent to the Gemini API. The model analyzes the prompt and generates a structured response in Markdown format.</li>
<li><b>Response Parsing:</b> The backend parses the Markdown response, extracting the content for each category (Symptoms, Causes, Cure, etc.).</li>
<li><b>History Logging:</b> The user's query and the AI's parsed analysis are saved as a new entry in the history.json file with a timestamp.</li>
<li><b>Display Results:</b> The parsed data is sent back to the frontend as a JSON object, where JavaScript dynamically populates the UI to display the analysis to the user.</li>
</ul>
